Fix issues in the KitchenOS codebase:

1. Ask me what needs to be fixed:
   - Specific error messages?
   - Bug description?
   - File or component with issues?

2. Investigate the issue:
   - Read relevant files
   - Check for TypeScript errors
   - Review recent changes if applicable

3. Create a plan to fix:
   - Identify root cause
   - Propose solution
   - List affected files

4. Implement the fix:
   - Make necessary changes
   - Ensure TypeScript types are correct
   - Maintain code style consistency

5. Verify the fix:
   - Run `npm run lint`
   - Test locally with `npm run dev`
   - Check for any side effects

6. Ask if I'd like to commit the fix
